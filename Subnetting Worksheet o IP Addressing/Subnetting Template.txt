- D2 must be the DHCP client

1. Convert CIDR to bits:
   Convert: 25 = 7 bits

2. Subtract bits from 32 to find subnet mask:
   Subtract: /32 - 7 = /25 (Octet, 128i)

3. Compute network IP:
   IPasok: 32i sa 4th octet = 192.168.50.0/25

4. Network Details:
   Network IP (Company) = 192.168.50.0 255.255.255.128
   Valid Range:
     - First Valid IP = 192.168.50.1
